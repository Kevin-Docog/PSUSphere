# PSUSphere

PSUSphere is a Django-based web application designed to manage student organizations within a university.  
It allows administrators to manage colleges, academic programs, student organizations, students, and their memberships through the Django Admin interface.

---

## Features

- Manage Colleges and Academic Programs
- Manage Student Organizations
- Manage Students and Organization Memberships
- Django Admin customization with search and filters
